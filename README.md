# Bank-Application--Group-Project

Banking application -- Group project for my C++ class 

- Customers with bank accounts can be added                   

- One of 6 transaction types can be processed:             
    1.Deposit                                                
    2.Withdrawal                                              
    3. Wire                                                    
    4. Cash a Check                                                   
    5. Transfer                                                
    6. Balance Inquiry
